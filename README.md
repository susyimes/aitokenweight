# aitokensflux Token 消耗计算器

一个海报式 Token 消耗量计算网页。输入今日 token、估算能耗参数和报告信息后，可以生成类似社交分享卡片的结果，并支持复制摘要和导出 PNG。

## 功能

- 输入 / 输出 / 缓存 token 拆分统计
- 按 Wh / 1K tokens 估算等效用电量
- 换算为手机充电、电动车续航、烧开水等生活化指标
- 根据 token 总量生成开发者百分位、等级和本周趋势
- 本地保存数据，支持保存今日、复制摘要、导出 PNG

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
